About

An iOS engineer who ships — and ships well.

I'm a native iOS developer from Brasília, Brazil, with 5+ years of experience building and scaling Apple-platform applications used by hundreds of thousands of people every day. I specialise in Swift, SwiftUI, and modern architecture patterns like MVVM, Clean Architecture and modularisation through Swift Package Manager.

I work fluently across iPhone, iPad, watchOS and tvOS. I've taken products from a blank Xcode project to the App Store in 1.5-week sprints, and I've also rescued legacy Objective-C codebases. I care a lot about clean code, accessibility, performance, and giving every user a great experience.

I was awarded an Apple WWDC Scholarship in both 2020 and 2021, and my work on the Poupatempo app (a public-services platform for the State of São Paulo) was recognised in the Public Sector category at the Agilidade Brasil 2024 Award.

Featured project

The thing I've been pouring the most love into.

Swift · SwiftUI · SPM · MVVM

JobFinder

A modular iOS app for finding and tracking job opportunities — built with SwiftUI, organised as a set of independent Swift Package Manager modules (ReduxCore, NetworkLayer, VacancyView, Components, MarketPulse) and shipped with a full CI/CD pipeline (Fastlane + Danger + SwiftLint).

The architecture is designed to be modular-by-default: every feature lives in its own package with clear boundaries, reusable UI components and a networking layer that can be tested in isolation. It is the codebase I use to demonstrate how I like to structure production iOS apps.

  • Architecture Modular SPM packages
  • UI SwiftUI
  • Concurrency async/await, Combine
  • CI/CD Fastlane, Danger, SwiftLint, GitHub Actions
  • Testing Unit tests per module
